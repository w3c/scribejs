Class Actions

Class to encapsulate all methods to handle actions. The public methods are:

  • set_date
  • add_action

This methods must be called in this order (with, of course, as many add_action calls as necessary).

set_date

  • set_date(date: string): void
    • Defined in src/lib/actions.ts:59

    Set the date of all actions. This method is called by the minute generator once the date has been established (e.g., from a 'date' line in the IRC log). Based on the date, this method sets the value of this.url, replacing the patterns in acurlpattern, if necessary.
